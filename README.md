# AI Chatbot
This is a project that randomly came to my mind... Through this project I will be able to learn things like different types of Backend Languages, how linking databases work and much more. This is going to be crazy fun because through this I will be able to achieve things like a working and talking Robot.
## How does this work?
There are many developers out there, and they are looking to be able to add a **Artificial Intelligence (AI)** to their project, since training AI Models can be a difficult and stressful task, other big AI companies released their things called **API key or a Application programming interface key**, which allows two different devices on the internet to talk to each other, in this case we have the user client PC and the AI Data center. Through API we can achieve a AI on our webpages.
## Why is learning this important?
By experimenting with this and making a AI, I will be able to improve a significant skill required for **Data Science**. As I make this AI Chatbot, I will be able to understand Databases and how API work.
